# Game_Maken
Klik op de groene knop boven aan de repo en download de ZIP.

Start het spel op door eerst de juiste versie van python te downloaden: python 13, dit kan bijvoorbeeld met Thonny of VS-code. Indien je VS-code gebruikt, vergeet dan niet python op VS-code te installeren. Installeer ook vanaf je terminal pygame en tkinter. Als je pip hebt op je laptop, gebruik dan
pip3 install pygame
pip3 install appJar
Als je homebrew hebt, gebruik dan
brew install pygame
brew install appJar
Je kunt pip en homebrew installeren vanaf de volgende websites:
https://pip.pypa.io/en/stable/installation/ 
https://brew.sh/

Als je de extensies hebt, kun je de ZIP file openen. In de ZIP file staan twee startpagina’s:  main.py en main_2048_CHAT_vers.py
Indien je in de klas zit, opent de docent voor je een van de twee versies, de docent zal de versies eerlijk verdelen over de klas, onthoudt welke versie je hebt om de enquête in te kunnen vullen.
Indien je thuis speelt, kies dan een van de twee versies om te spelen. Je kunt hierna ook de andere versie uitproberen. Vul dan alsjeblieft de enquête tweemaal in.

# # Hoe te spelen:
# Quiz
klik vanuit de het quiz beginscherm op de “vragen” knop, de vragen zullen gelijk beginnen, je hebt 15 seconden voor elke vraag. Er zijn 3 antwoorden om uit te kiezen waarvan 1 juist is. Na het antwoorden kom je te weten of je gekozen antwoord goed is. Na de quiz kom je in een eindscherm met jouw score (goed antwoord is +1 punt, fout antwoord -1 punt). 

# 2048
Het klassieke spel 2048 kent iedereen! Indien je dit niet kent, hier is de korte versie, opgedeeld in de verschillende versies van het spel.
main.py
Gebruik de pijltjes-toetsen of wasd-toetsen om de blokken te bewegen. Als dezelfde blokken botsen, dan zullen zij zich samenvoegen tot de optelsom. Zo wordt 2+2 4,4+4 8, 8+8 16, etc.
Het doel van het spel is om 2048 te maken. Na elke beurt komt er een nieuw blokje met 2 of 4 erop in het veld. Als je niets meer kan omdat het veld vol is en niets meer kan worden samengevoegd, heb je verloren!
Veel plezier!
main_2048_CHAT_vers.py
Gebruikt de wasd-toetsen om de blokken te bewegen. Als dezelfde getallen botsen, dan zullen zij zich samenvoegen tot de optelsom. Zo wordt 2+2 4,4+4 8, 8+8 16, etc.
Het doel van het spel is om 2048 te maken. Na elke beurt komt er een nieuw getal (2 of 4) in het veld. Als je niets meer kan omdat het veld vol is en niets meer kan worden samengevoegd, heb je verloren!
Veel plezier!

# PIG
Het spel PIG is een kansspel. Let op! De window waarin PIG staat, kan achter het hoofdscherm worden geplaatst. In dat geval kun je het hoofdscherm simpelweg verplaatsen en op het PIG window klikken om te spelen. 
Voer eerst in met hoeveel spelers je speelt (2 tot 4). Klik vervolgens op de knop om het spel te starten.
Speler 1 zal beginnen. Als je aan de beurt bent kun je twee dingen kiezen: roll of hold.
Hold betekent dat je beurt klaar is en je punten worden opgeslagen, deze kun je nu niet meer verliezen.
Roll betekent dat je een digitale (ontzichbare) dobbelsteel rolt. Je punten zijn de ogen van de dobbelsteen. Je mag zo vaak rollen als je wilt, en alle punten worden opgeteld. Maar let op! Zodra je een 1 gooit, gaan je punten die je niet hebt opgeslagen, verloren. Je beurt is ook voorbij en speler 2 is aan zet!
Als alle spelers hun beurt hebben gehad, noem je dat een ronde. 
Zodra een speler meer dan 150 punten in totaal heeft gehaald en deze heeft opgeslagen met Hold, mogen de andere spelers hun ronde afmaken. De speler met de meeste punten heeft dan gewonnen!
